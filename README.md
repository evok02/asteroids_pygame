# Asteroids PyGame

A modern implementation of the classic Asteroids arcade game built with Python and Pygame. 


## Installation

### Prerequisites
- Python 3.10 or higher
- Pygame 2.0 or higher

### Quick Start
```bash
# Clone the repository
git clone https://github.com/evok02/asteroids_pygame.git
cd asteroids_pygame

# Install dependencies
pip install pygame

# Run the game
python main.py
